# Deploy Spendly on Render

## 1. Push this project to GitHub
Create a repository and push this folder contents.

## 2. Create service on Render
1. Open Render Dashboard.
2. Click **New +** -> **Blueprint**.
3. Connect your GitHub repository.
4. Render will detect `render.yaml` automatically.
5. Click **Apply**.

## 3. First run behavior
- `app.py` calls `init_db()` automatically at startup.
- SQLite file is stored at `/var/data/expense_tracker.db` using the mounted Render disk.

## 4. Environment variables
`render.yaml` already sets:
- `FLASK_ENV=production`
- `SPENDLY_SECRET_KEY` (generated by Render)
- `SPENDLY_DB_PATH=/var/data/expense_tracker.db`

## 5. Access your app
After deployment is healthy, open the Render URL and register your first user.
